MUKAGEN

Modular Counter-UAS Weapon Station

Multi-sensor fusion

The EV-TINYRAD24G 24GHz FMCW radar handles initial acquisition, providing range, velocity, and bearing on inbound targets before the camera ever activates. YOLOv8n confirms the visual track; Florence-2 classifies threat type and behavior. An Extended Kalman Filter fuses all three streams into a single continuously updated threat picture — eliminating the false positives that trip single-sensor systems.

Edge-deployed AI

All inference runs on the Jetson Orin NX — no cloud calls, no network dependency, no added latency. YOLOv8n runs TensorRT INT8-quantized for sub-millisecond detection; Florence-2 handles threat classification entirely on-device. Mukagen operates in GPS-denied and comms-denied environments without degradation.

Modular effector interface

The weapon station exposes a standardized payload interface, allowing operators to configure RF jamming, directed energy, or kinetic effectors without hardware redesign. Effector engagement is gated behind deterministic, rule-based threat confirmation — every engagement decision is auditable and reproducible.

3-axis pan/tilt/roll gimbal

Full pan/tilt/roll articulation enables continuous target lock across azimuth, elevation, and roll — maintaining track through aggressive target maneuvers and platform vibration. High-resolution servo control minimizes settling time during intercept handoff.

Configuration A — Fixed-Site Defense

Optimized for perimeter hardening of static assets. Deployed on ground mounts or rooftop hardpoints with 360° radar sweep and sector-based EO tracking. Supports continuous unattended operation; operator intervention required only at fire authorization.

Configuration B — Vehicle-Mounted

Compact form factor for integration on light tactical vehicles. Rail-mount interface supports rapid forward deployment and redeployment. Full system capability retained in mobile configuration with no software changes required.

Operational deployment

Mukagen initializes autonomously and achieves full radar coverage within seconds of power-on. No network connectivity required at any stage of operation. Threat tracks surface through the Argus GCS with operator-in-the-loop authorization before any effector command is issued. Designed for base perimeter defense, VIP protection zones, and forward operating base hardening.

Ground tripod

Portable standalone deployment on level terrain. Fully self-contained with no external power or structure required.

Vehicle rail mount

NATO STANAG-compatible roof bracket for integration on light tactical vehicles. Rapid forward deployment and redeployment without tooling.

Fixed hardpoint

Structural bolt pattern for permanent installation on concrete or steel structure. Intended for long-duration perimeter defense of static assets.
